My React Calendar

My React Calendar(MRC) is a simple React Component to display events on calendar view.

MRC is using TailwindCSS, if your project are also using TailwindCSS, you need to add MRC to your purge array. 


module.exports = {
    ...
        purge: [
            "./src/**/*.{js,jsx,ts,tsx}",
            "./public/index.html",
            "./node_modules/my-react-calendar/dist/components/**/*.{js,jsx}"
        ],
    ...
}

        
            
import Calendar from "my-react-calendar";

export default function Wrapper() {

    return (
        <Calendar
            events={eventsData}
            use24Hour={false}
            enableTimezone={true}
            status={fetchStatus}
            onChange={onUserChange}
        />
    )
}
